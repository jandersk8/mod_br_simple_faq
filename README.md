# BR Simple FAQ Module for Joomla 5 & 6

A modern, lightweight, and SEO-friendly FAQ accordion module for Joomla. This module was built with performance in mind, using vanilla JavaScript and CSS (no jQuery or Bootstrap dependencies required).

## ğŸš€ Features (English)

* **10 FAQ Items**: Support for up to 10 question and answer pairs (5 items are pre-filled with Lorem Ipsum for quick testing).
* **Fully Responsive**: Works perfectly on desktops, tablets, and smartphones.
* **Modern Design**: Clean accordion effect with smooth CSS transitions.
* **SEO Optimized**: Built-in **Schema.org (FAQPage)** microdata to help search engines display your FAQs directly in search results (Rich Snippets).
* **High Customization Power**:
    * Custom colors for titles/questions (normal and active/open states).
    * Custom background colors for answers (transparent support).
    * Adjustable border colors and border-radius (rounded corners).
    * Font size control for both questions and answers.
* **SVG Icons**: Lightweight, high-quality vector icons that scale perfectly without external fonts.
* **Single Open Logic**: Automatically closes the previous item when a new one is clicked to keep the layout organized.
* **Joomla 6 Native**: Built using modern Namespaces and Service Providers for maximum performance.
